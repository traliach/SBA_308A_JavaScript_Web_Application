# Manga Hub

A single-page JavaScript web app that searches manga (Jikan) and saves a personal list (CrudCrud).

## Run
Open `index.html` with Live Server (recommended) or in the browser.

## Setup (CrudCrud)
1. Copy `js/config.example.mjs` to `js/config.mjs`
2. Put your CrudCrud base URL inside `CRUDCRUD_BASE`
